# Мем-хостинг (Платформа для публикации картинок)

Учебный проект по летней практике.

## 🚀 Описание проекта
Сайт, куда пользователи могут загружать картинки, мемы и писать посты, указывать свой ник, ставить теги и лайкать чужие посты. Все посты красиво выводятся на главной странице в виде удобной сетки.

## 📐 Архитектура проекта
Проект разделен на 3 части, которые запускаются одновременно через **Docker Compose**:
1. `frontend` — визуальная часть сайта на **Vue.js 3** (интерфейс, сетка постов, кнопки лайков).
2. `backend` — серверная часть на **Python (FastAPI)**, которая принимает картинки и обрабатывает логику.
3. `db` — готовая база данных **PostgreSQL** для хранения информации о постах, лайках и тегах.

## 📦 Docker Volumes
Чтобы мемы и лайки не удалялись при перезапуске контейнеров, мы настроили сохранение файлов на жесткий диск компьютера:
- `pgdata` — сохраняет базу данных (текст постов, лайки, ники авторов).
- `media_data` — сохраняет сами файлы картинок, которые загружают пользователи.

## 🛠 Предварительные требования для запуска
- Docker Engine / Docker Desktop
- Docker Compose
